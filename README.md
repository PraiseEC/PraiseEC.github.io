## 🚀 Projects

Here are some of the projects featured on my portfolio:

### 🔹 Project 1 – [Voice-Contrlolled-Home-Automation-Robot-System]
- Description: This project enables voice-controlled automation of lights, appliances, and a simulated door using an Arduino Uno, relay modules, and a servo motor. Commands are sent via a serial interface (e.g.Bluetooth terminal) to trigger actions like turning devices on/off or opening/closing a door.    
- Tech Used: C++  
- GitHub Repo: https://github.com/PraiseEC/Group-7-Mechatronics-Voice-Contrlolled-Home-Automation-Robot-System

- Description: A modern, high-performance desktop application designed to solve the problem of digital clutter. This tool uses a rule-based algorithm to sort files into meaningful categories instantly.
- Tech Used:  Python
- GitHub Repo:  https://github.com/PraiseEC/File-Organizer-Pro

- Tech Used:  MATLAB, SIMULINK, TINKERCAD
