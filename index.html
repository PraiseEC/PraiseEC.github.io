Project 1

## 📝 Description This project enables voice-controlled automation of lights, appliances, and a simulated door using an Arduino Uno, relay modules, and a servo motor. Commands are sent via a serial interface (e.g.Bluetooth terminal) to trigger actions like turning devices on/off or opening/closing a door. ## 🚀 Features - Voice command control for 3 relays (appliances/lights) and 1 servo motor (door). - Real-time feedback via LED - Group commands (`all on`/`all off`) for bulk control. - Modular code structure for easy customization. ## 🛠️ Hardware Requirements - Arduino Uno - 5V Relay Modules (x3) - SG90 Servo Motor - Jumper Wires & Breadboard - LED (for status feedback) - 1kΩ Resistor (for LED) - USB Cable (for Arduino) - Device for sending commands (PC/smartphone with serial terminal app)

Project 2

A modern, high-performance desktop application designed to solve the problem of digital clutter. This tool uses a rule-based algorithm to sort files into meaningful categories instantly. 🚀 Key Features Modern UI: Built with CustomTkinter featuring a responsive Dashboard and Dark/Light mode. Auto-Sorting: Categorizes files into Images, Documents, Videos, Music, and Archives. Real-time Monitoring: Uses Watchdog to detect new downloads and sort them automatically. Data Visualization: Generates interactive Pie Charts using Matplotlib to analyze storage usage. Smart Tools: Includes a Duplicate File Finder, Batch Renamer, and Empty Folder Cleaner. Safety First: Features a comprehensive "Undo" function and activity logging.
